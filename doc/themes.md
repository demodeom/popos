# 主题美化

## Gnome Shell Extensions

1. 安装 gnome-shell-extensions

```
sudo apt install gnome-shell-extensions
```

2. 注销系统， 重新登陆

3. 使用 Gnome Shell Extension Manager 软件，打开 **User Themes** 扩展


## ocs-url

ocs url 是一款支持自动安装 [https://www.pling.com/](https://www.pling.com/) 主题的工具

ocs url 下载地址 [https://www.opendesktop.org/p/1136805/](https://www.opendesktop.org/p/1136805/)


```
sudo dpkg -i ocs-url_3.1.0-0ubuntu1_amd64.deb
```

如果安装出错，可使用以下命令自动解决依赖

```
sudo apt install -f
```
