## LocalSend

局域网聊天文件传输

```bash
cd ~/Downloads

wget https://ghproxy.net/https://github.com/localsend/localsend/releases/download/v1.14.0/LocalSend-1.14.0-linux-x86-64.deb

sudo dpkg -i ~/Downloads/LocalSend-1.14.0-linux-x86-64.deb

sudo apt install -f -y
```
